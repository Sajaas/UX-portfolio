Chubby Hamster Knight

My first video game!

I created a 2D platformer in order to learn how Unity works. It was fun to create 2D art, code, and design levels and the story!

It has a working game loop, 5 levels, and a simple boss fight.

Reason

To learn Unity I decided to create a short 2D pixel platformer. The music is made by my friend, and the rest by me!

My roles

Since it's a hobby project, and my first video game, I ended up learning a bit about everything.

I set the up the project, programmed, designed levels, enemies, obstacles, goals, story, UI. I even created the pixel art and animations for the character and the villain, an experience which also was new to me!

Setting

2D Pixel Platformer in a fantasy world where each level has a new enviroment and interactions.
