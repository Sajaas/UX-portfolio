Chubby Hamster Knight

Hobby project to learn Unity and C#

try the game
Reason

To learn Unity I decided to create a short 2D pixel platformer, with support from a friend.

My roles

A bit of everything. I set the up the project, programmed, designed levels, enemies, obstacles, goals, story, UI... I created the pixel art and animations for the character and the villain.

Setting

Platformer in a fantasy world where each level has new enviroment and interactions.
